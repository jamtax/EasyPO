Important conversion notes

  • Embedded text PDFs: These generally produce the fastest and cleanest results because the text layer can be read directly.
  • Scanned PDFs: OCR improves recoverability, but accuracy depends on scan quality, skew, contrast, and language support. This version defaults to English OCR.
  • Tables: The table engine uses spatial heuristics from PDF coordinates. It performs well on invoices, statements, schedules, and tabular reports, but irregular visual layouts may still require cleanup.
  • Performance: OCR is CPU-intensive. Large or image-heavy PDFs can take significantly longer, especially on low-power devices.
  • Privacy: Processing is local-first in this static implementation. No remote storage workflow is included here.
